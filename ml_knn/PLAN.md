# KNN 算法子项目计划文档

## 📌 基础信息

- **算法名称（原始）：** KNN
- **算法名称（规范化）：** knn
- **分类：** 机器学习 (ML)
- **分类理由：** KNN (K-Nearest Neighbors) 是经典的基于实例的机器学习算法，使用距离度量进行分类/回归，不涉及神经网络训练，属于传统机器学习范畴。
- **创建日期：** 2026-01-18
- **子项目路径：** `ml_knn/`

---

## ⚠️ 重要：虚拟环境依赖

**本项目是 `MLDLAlgorithmCode` 总仓库的子项目，依赖于统一的虚拟环境 `mldl_algo311`。**

**所有操作必须在该虚拟环境下进行：**

```bash
# 激活虚拟环境
conda activate mldl_algo311

# 进入项目根目录
cd MLDLAlgorithmCode

# 进入 KNN 子项目目录
cd ml_knn

# 运行项目
python main.py
```
