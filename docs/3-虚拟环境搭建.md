## ✅ MLDLAlgorithmCode 专用虚拟环境（极简一行版）

### 1️⃣ 创建并激活环境（Python 3.11）

```powershell
conda create -n mldl_algo311 python=3.11 -y && conda activate mldl_algo311
```

---

### 2️⃣ 安装 GPU 版 PyTorch（CUDA 12.1）

```powershell
pip install torch==2.5.1 torchvision==0.20.1 torchaudio==2.5.1 --extra-index-url https://download.pytorch.org/whl/cu121
```

---

### 3️⃣ 安装算法仓库基础依赖

```powershell
pip install numpy scipy pandas matplotlib scikit-learn tqdm pyyaml
```

---

### 4️⃣ 验证 GPU 是否可用

```powershell
python -c "import torch; print(torch.__version__, torch.cuda.is_available(), torch.cuda.get_device_name(0) if torch.cuda.is_available() else None)"
```

---

## ✅ 日常 Conda 使用命令（你只需要这些）

### 查看已有虚拟环境

```powershell
conda env list
```

---

### 激活 MLDLAlgorithmCode 环境

```powershell
conda activate mldl_algo311
```

---

### 退出当前虚拟环境

```powershell
conda deactivate
```

---

### 查看当前环境安装的包

```powershell
pip list
```

---

### 查看 PyTorch + GPU 是否可用

```powershell
python -c "import torch; print(torch.__version__, torch.cuda.is_available())"
```

---

### 查看 GPU 信息（可选）

```powershell
nvidia-smi
```

---

### 删除虚拟环境（慎用）

```powershell
conda remove -n mldl_algo311 --all -y
```

---

## ✅ 你之后的“固定使用流程”（记住这 2 行就够）

```powershell
conda activate mldl_algo311
cd MLDLAlgorithmCode
```

从现在开始，你**只需要这套命令**就能稳定推进你的算法项目了。
